# שומרי התבלין — MVP רזה

משחק Web סטטי, מובייל־פירסט, ללא שרת וללא תלויות.

## קבצים
- `index.html` — המשחק
- `style.css` — עיצוב
- `data.js` — תחנות, דמויות, שאלות, שפות
- `app.js` — לוגיקת המשחק
- `master.html` — דף קישורים ליצירת QR
- `assets/characters/` — תמונות הדמויות

## העלאה
להעלות את כל התיקייה ל־Netlify / GitHub Pages / שרת סטטי.
לא לפתוח רק את `index.html` מהטלפון, כי אז תמונות עלולות להישבר.

## קישורי תחנות
לאחר העלאה, פתחו:
`/master.html`
והעתיקו משם את הקישורים.

מבנה הקישורים:
- `/?station=tatzpit`
- `/?station=shomera`
- `/?station=beitbad`
- `/?station=gat`
- `/?station=goren`
- `/?station=bor`

## צרפתית
הממשק מתורגם לצרפתית. כרגע שאלות התוכן בצרפתית נופלות לעברית עד שננסח גרסה צרפתית מלאה.
ניתן לפתוח עם:
`/?lang=fr`
או תחנה בצרפתית:
`/?station=gat&lang=fr`

## מצב בדיקה
הוסיפו `?dev=1` לכתובת כדי להציג כפתורי בדיקה פנימיים במפת ההתקדמות.
לא להשתמש בזה בהפעלה מול שחקנים.

## בדיקות חובה לפני הדפסה
1. לפתוח משחק חדש.
2. לוודא שמסלול התחנות אקראי ונשמר.
3. לפתוח קישור תחנה נכונה — מופיע מסך מפגש.
4. לפתוח קישור תחנה לא נכונה — מופיעה הודעה משחקית.
5. לבחור כל מסיח בכל שאלה — החיווי חייב להתאים למסיח שנבחר.
6. להשלים שש תחנות — מופיע מסך סיום.
7. לבדוק בטלפון ברוחב מסך רגיל.
